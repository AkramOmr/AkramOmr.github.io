Les Réseaux Informatiques

Introduction :

Il est fort probable que la première chose que vous faisiez après avoir reçu un nouvel ordinateur vous essayerai de vous connecter à internet pour naviguer en ligne, entre temps vous établissez une connexion qui va appelez votre modem à échanger les informations via des signaux analogiques. Ce processus peut se projeter sur tous les appareils et ordinateurs du monde, ce qui forme une chaîne de connexion qu’on appelle réseaux informatiques.
